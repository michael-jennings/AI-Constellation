## Quick start

```bash
npm install
npm run seed
npm run dev
```

## Architecture

- **Next.js App Router** in `/app`
- **Tailwind CSS** for styling
- **Zustand** for client state
- **Synthetic data** in `/data` generated by `npm run seed`
- **Recharts** for charts
- **Vitest** for unit tests

## Modifying scoring
Weights live in `lib/scoring.ts`. Adjust the coefficients for recency, cost, latency, fit, adoption, and credibility as desired.
